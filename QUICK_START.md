# クイックスタートガイド

このガイドでは、最短でアプリを動作させる方法を説明します。

## 📋 前提条件

以下がインストールされていることを確認してください：

- Node.js (v18以上推奨)
- npm または pnpm

## 🚀 3ステップで起動

### ステップ1: 依存関係のインストール

```bash
npm install
```

または

```bash
pnpm install
```

### ステップ2: 開発サーバーの起動

```bash
npm run dev
```

または

```bash
pnpm dev
```

### ステップ3: ブラウザで確認

ブラウザで以下のURLを開いてください：

```
http://localhost:5173
```

✅ これで完了です！アプリが動作しているはずです。

---

## 📱 モバイルでの確認方法

### 同一ネットワークでのテスト

1. 開発サーバーを起動すると、コンソールに以下のようなメッセージが表示されます：

```
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.1.100:5173/
```

2. スマートフォンのブラウザで `Network` のURLにアクセスしてください。

### QRコードでアクセス（推奨）

ターミナルで以下を実行（事前にqrcodeパッケージをインストール）：

```bash
npx qrcode http://192.168.1.100:5173
```

表示されたQRコードをスマートフォンでスキャンしてください。

## 🛠️ カスタマイズの最初の一歩

### 1. パワーワードを変更する

`/src/app/data/powerwords.json` を開いて、任意のワードを編集：

```json
{
  "id": 1,
  "rank": "神級",
  "word": "「あなたのオリジナルワード」",
  "source": "第X章 X節, p.XX",
  "isRare": true
}
```

保存すると、ホットリロードで自動的に反映されます。

### 2. 書籍購入リンクを設定する

`/src/app/components/ResultScreen.tsx` の55行目あたり：

```typescript
const handlePurchase = () => {
  // このURLを実際の購入ページのURLに変更
  window.open('https://techbookfest.org/product/YOUR_BOOK_ID', '_blank', 'noopener,noreferrer');
};
```

### 3. 神級の出現率を変更する（テスト用）

`/src/app/App.tsx` の17行目：

```typescript
// 1%から50%に変更（テストしやすくする）
const selectedWord = drawPowerWord(0.5);
```

**注意**: 本番環境にデプロイする前に、必ず元の `0.01` に戻してください！

## 🎨 デザインの確認

### 通常のワードをテストする

1. おみくじを何回か引く
2. 通常級（アーキテクト級、リードエンジニア級など）が表示される
3. ブルーの配色とシンプルなアニメーションを確認

### 神級をテストする

神級が出るまで待つのは大変なので、一時的に確率を上げてテスト：

```typescript
// App.tsxで確率を100%に
const selectedWord = drawPowerWord(1.0);
```

これで必ず神級が出ます：
- 黄金のグラデーション背景
- 紙吹雪エフェクト
- キラキラパーティクル

確認後、必ず `0.01` に戻してください！

## 🆘 よくある問題

### Q: `npm install` でエラーが出る

**A**: Node.jsのバージョンを確認してください：

```bash
node --version
```

v18以上が必要です。アップデートしてください。

### Q: 画面が真っ白

**A**: ブラウザのコンソールを開いて、エラーメッセージを確認してください：

- Chrome: F12 → Console タブ
- Safari: 開発メニュー → JavaScriptコンソール

### Q: おみくじを引いても何も起こらない

**A**: 
1. コンソールにエラーが出ていないか確認
2. `powerwords.json` の構文エラーをチェック
3. ブラウザをリフレッシュ（Ctrl+R / Cmd+R）

### Q: モバイルでアクセスできない

**A**: 
1. PCとスマホが同じWi-Fiに接続されているか確認
2. ファイアウォールがブロックしていないか確認
3. VPNを使用している場合は一時的にオフにする

## 💡 Tips

### 開発中のホットリロード

Viteは自動でファイルの変更を検出して、ブラウザを更新します。
保存するだけで、すぐに変更が反映されます！

### 複数のワードを一度に追加する

`powerwords.json` は配列なので、以下のように複数追加できます：

```json
[
  { "id": 1, ... },
  { "id": 2, ... },
  { "id": 3, ... }
]
```

**重要**: 最後の項目の後にカンマを入れないでください（JSONの仕様）。

### パフォーマンスチェック

Chrome DevToolsのLighthouseを使用して、パフォーマンスをチェック：

1. F12でDevToolsを開く
2. Lighthouse タブ
3. 「Generate report」

スコア90以上を目指しましょう！

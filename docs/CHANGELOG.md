# Changelog

このプロジェクトの変更履歴を記録します。

## [1.0.0] - 2026-03-23

### 追加機能

- ✨ パワーワードおみくじの基本機能を実装
- ✨ ホーム画面（書籍情報とおみくじボタン）
- ✨ 結果表示画面（パワーワード表示）
- ✨ ローディング画面（抽選中のアニメーション）
- ✨ 52個のパワーワードデータを実装
- ✨ 神級（レア）の特別演出を実装
  - 黄金のグラデーション背景
  - 紙吹雪エフェクト
  - キラキラパーティクル
  - 回転する光の輪
- ✨ SNS共有機能（X/Twitter）
- ✨ 書籍購入リンク
- ✨ レスポンシブデザイン（モバイルファースト）

### デザイン

- 🎨 書籍のブランドカラー（ブルー＆グレー）を採用
- 🎨 回路図パターンの背景デザイン
- 🎨 エンジニアリングと文学を融合したデザインコンセプト
- 🎨 Motion（Framer Motion）による滑らかなアニメーション

### 技術実装

- ⚡ React 18.3.1 ベース
- ⚡ TypeScript による型安全性
- ⚡ Tailwind CSS v4 でスタイリング
- ⚡ Vite でビルド最適化
- ⚡ レア確率1%の抽選ロジック
- ⚡ ユーティリティ関数による抽選ロジックの分離

### ドキュメント

- 📚 README.md - プロジェクト概要
- 📚 USAGE_GUIDE.md - 詳細な使用方法
- 📚 DEPLOYMENT_NOTES.md - デプロイガイド
- 📚 index.html.example - HTMLメタタグのサンプル

### データ

- 📊 52個のパワーワード
  - 神級: 2個（1%の出現率）
  - アーキテクト級: 17個
  - リードエンジニア級: 17個
  - シニアエンジニア級: 16個

## 変更履歴の記述ルール

このプロジェクトでは、以下の絵文字を使用してカテゴリを区別します:

- ✨ 新機能
- 🐛 バグ修正
- 🎨 デザイン変更
- ⚡ パフォーマンス改善
- 📚 ドキュメント
- 🔒 セキュリティ
- 💥 破壊的変更
- 📊 データ更新
- 🚀 デプロイ関連

バージョン番号は [Semantic Versioning](https://semver.org/) に従います:
- MAJOR: 破壊的変更
- MINOR: 後方互換性のある機能追加
- PATCH: 後方互換性のあるバグ修正
